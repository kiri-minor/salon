# 💆‍♀️ L'ORAGE — Лендинг салона красоты

Одностраничный лендинг премиум-салона красоты. Мятно-белая гамма, воздушные анимации и атмосфера спа.

## ✨ Особенности

- 🍃 **Мятно-белая палитра** — свежесть, чистота и расслабление
- 🌊 **Плавные анимации** — волны, появление из тумана, пульсации
- 💎 **Премиум-секции** — услуги, мастера, прайс, отзывы
- 📸 **Галерея работ** — слайдер с эффектом увеличительного стекла
- 📅 **Форма записи** — выбор услуги, мастера и времени
- 📱 **Полная адаптивность** — mobile-first подход
- 🎯 **SEO-оптимизация** — мета-теги, Open Graph, schema.org

## 🛠 Технологии

| Технология | Назначение |
|-----------|-----------|
| **HTML5** | Семантическая вёрстка |
